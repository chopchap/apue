## syslog(3)
- De-factor standard for message logging on Unix systems since the 1980s.
- Documented in RFC3164, now standardized in RFC5424.
- On the network, defaults to UDP port 514.
- Various newer versions with many new features (TCP, TLS, logging to a database or a filter, ...) available, e.g., syslog-ng, rsyslog.
- Quickly leads to large-scale messaging systems including e.g., Elasticsearch, Solr, Flume, Fluentd, ...
